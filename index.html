Clase 2 - Ejercicio 1

Tengo desarrollada la siguiente función en javascript

                    function suma (a,b,callback) {
                        let c = a +b;
                        callback();
                    }
  1. ¿Que es el tercer parámetro recibido?

    2. El tercer parametro recibido es una función, por ejemplo, en este caso puede ser la funcion que permite mostrar en pantalla variable "c" si se la pasa como parametro en "callback()".

  2. ¿En que casos es obligatorio desarrollar este tipo de funciones?

    3. Este tipo de funciones se utilizan para modularizar el codigo y lograr ahorrarnos las escritura de codigo repetido como por ejemplo mostrar por pantalla el resultado de una suma, una resta, una multiplicacion o una division.
